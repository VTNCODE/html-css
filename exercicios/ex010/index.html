Usando links externos

Você pode acessar o meu repositório público do Github

Perceba a diferença entre o link de cima para o de baixo, o link acima foram usadas os parâmetros target= "blank" e rel="external". Esses dois parâmeteos fizeram com que o link abrisse em uma guia externa, não fechando o site.

Agora olhe o de baixo, sem os parâmetros citados acima, o link contiará abrindo, mas dessa vez em sobrepõe o seu site, fazendo o que feche.
